Projeto para fins de portfólio front-end júnior.

# Hamburgueria JS

Projeto de e-commerce front-end para hamburguerias e delivery.

# Funcionalidades
- Carrossel de produtos
- Adicionar/remover itens do carrinho
- Cálculo automático do total
- Carrinho persistente (LocalStorage)
- Checkout via WhatsApp

# Tecnologias
- HTML
- CSS
- Bootstrap 5
- JavaScript puro
