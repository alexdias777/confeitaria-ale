# Buscador de CEP

Aplicação front-end desenvolvida com **HTML, CSS e JavaScript** que realiza a consulta de endereços a partir de um CEP, utilizando uma **API pública**.

---

## Funcionalidades

- Buscar endereço a partir do CEP
- Validação de CEP (8 dígitos)
- Consumo de API pública (**ViaCEP**)
- Exibição de mensagens de erro para CEP inválido ou não encontrado

---

## Tecnologias utilizadas

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- API ViaCEP

---

##

- Consumo de API com `fetch`
- Trabalhar com dados em formato JSON
- Manipulação do DOM
- Tratamento de erros em requisições
- Boas práticas em JavaScript básico
